## Для вывода кода использовать `````` (с Enter'ом между 3 и 4)
```
ВЫДЕЛЕННЫЙ КОД ЛОЛ
```
## Для создания заголовка разных уровней использовать # (есть 6 уровней)
# **ЭТО ТЕКСТ 1 ТИПА**
## **ЭТО ТЕКСТ 2 ТИПА**
### **ЭТО ТЕКСТ 3 ТИПА**
#### **ЭТО ТЕКСТ 4 ТИПА**
##### **ЭТО ТЕКСТ 5 ТИПА**
###### **ЭТО ТЕКСТ 6 ТИПА**
```
# ТИПО ЗАГОЛОВОК 1 УРОВНЯ
```
## использовать * для разных выделений
#### *наклонный*
#### **жирный**
#### ***жирный и наклонный***
```
*наклонный*
**жирный**
***жирный и наклонный***
```
